# docker-python-GTSN-game

```
Requirements:
{
	1. Allow player to enter credentials for authentication/authorization
	2. CSV file contains list of song names and artists
	3. User selects a song from the file, displaying the artist and the first letter of each word of the song title.
	4. Guess the song.
	5. If the guess is correct, the first time, 3 points.
	6. If the guess is correct the second time, 1 point.
	7. If the guess is wrong second time, the game ends.
	8. Validate User inputs
	9. If user successfully guessed the selected song, remove the song from the list and begin choosing again
}
```
